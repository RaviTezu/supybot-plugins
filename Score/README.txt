This for "Cricket" lovers. This plugins gets your the latest score.

Plugin Requirements:
   1. urllib2 python module.
   2. BeautifulSoup python module. 

Usage: 1. Use @load Score(This has to be done in chat channel or Private message your chat bot) command to load the plugin.
       2. @score to get the latest score. I use "@" symbol to instruct my chat bot, this may differ in your case. 
       3. As I'm from India, This gets you the score only when there's a India Match. You can change this to your Country in plugin.py file.
   
Note: This plugin is still in progress.
